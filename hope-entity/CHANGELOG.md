# Changelog

A Hope Entity minden változása dokumentálva.

## [0.1.0] - 2024

### Hozzáadva
- Entitás alaprendszer személyiséggel és érzelmekkel
- Ollama HTTP bridge több modell támogatással
- Natív GGUF inference (opcionális feature)
- Silent Teacher etikai keretrendszer (3 axióma)
- Önjavító rendszer (self-repair)
- Benchmark modul
- CLI eszköz (remeny)
- Magyar nyelvű kommunikáció
- Rövid és hosszú távú memória
- Kód generálás képesség
- Fordítás képesség

### Modellek támogatása
- Magyar: OpenEuroLLM-Hungarian
- Kódolás: DeepSeek Coder
- Többnyelvű: Qwen2.5

### Features
- `native` - GGUF modell beolvasztás
- `cuda` - NVIDIA GPU gyorsítás
- `metal` - Apple Silicon gyorsítás
